# Admin Assist

A brief description of whatThe core aim was to design and develop a web-based Student Information System, Admin Assist, to improve the accuracy, accessibility, and overall management of student records. The specific objectives guided the entire proposal: to analyze current manual practices, design a centralized database, develop core functional modules, implement secure access controls, and establish a framework for evaluating the system's effectiveness this project does.•	Sprint 1: User Authentication and Role-Based Access Control (RBAC). •	Sprint 2: Student Registration and Profile Management. •	Sprint 3: Attendance Recording and Management. •	Sprint 4: Academic Record Storage and Results Processing. •	Sprint 5: Report Generation and Data Retrieval.
